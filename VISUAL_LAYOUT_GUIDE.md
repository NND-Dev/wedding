## 📱 Responsive Layouts

### Mobile Layout (< 600px)
- Single column layout
- Stacked sections
- Smaller fonts and spacing
- 2-column gallery grid
- Full-width cards
- Form and QR code stacked vertically

### Desktop Layout (≥ 600px)
- Wider margins (48px vs 16px)
- Larger fonts
- 3-4 column gallery grid
- Side-by-side form and QR code
- Enhanced spacing between sections

## ✨ Animations

### Hero Section
```
[Closed Card]  →  [Opening 3D Flip]  →  [Opened Card]
    0s              1s - 2.5s              2.5s+
                                        ↓
                                   [Fade in Names]
                                   [Fade in Text]
                                   [Scale Date]
```

### Timeline Section
```
Item 1:  Dot appears (300ms) → Date slides (400ms) → Title slides (500ms) → Description slides (600ms)
Item 2:  Dot appears (500ms) → Date slides (600ms) → Title slides (700ms) → Description slides (800ms)
Item 3:  Dot appears (700ms) → Date slides (800ms) → Title slides (900ms) → Description slides (1000ms)
...
```

### Gallery Section
```
Title:       Fade in + Slide down (0ms)
Categories:  Fade in + Slide right (300ms)
Images:      Staggered fade in (300ms + 50ms per item)

Click:       Grid thumbnail → Lightbox (Hero animation)
```

### Message Section
```
Form card:   Fade in + Slide left (300ms)
QR card:     Fade in + Slide right (400ms)
```

## 🎯 Interactive Elements

### Wedding Details Cards
```
[Get Directions] → Opens Google/Apple Maps
[Copy Address]   → Copies to clipboard + shows confirmation
```

### Gallery
```
[Category Tab]   → Filters images by category
[Image]          → Opens lightbox viewer
  Lightbox:
    [Swipe/Arrow] → Navigate images
    [Pinch/Zoom]  → Zoom image
    [X/Tap]       → Close lightbox
```

### Message Form
```
[Name Input]     → Required validation
[Message Input]  → Required + min 10 chars
[Submit]         → Validation → Submit → Success message → Clear form
```
